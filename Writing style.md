# 写作规范

## 标题
标题分为三级
- 一级：文章的标题
- 二级：文章主要部分的大标题
- 三级：二级标题下一级小标题
    - **列表**：三级下的小标题

示例：
```markdown
# 一级标题
## 二级标题
### 三级标题
    - 列表
```
## 标点符号
- 中文全角（占两个字符）
- 英文半角（占一个字符）
- 中英混排，全角。英文整句中标点用半角。
