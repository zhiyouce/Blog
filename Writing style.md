# 写作规范

## 标题
标题分为三级
- 一级：文章的标题
- 二级：文章主要部分的大标题
- 三级：二级标题下一级小标题
    - **列表**：三级下的小标题

示例：
```markdown
# 一级标题
## 二级标题
### 三级标题
    - 列表
```
## 标点符号
- 中文全角（占两个字符）
- 英文半角（占一个字符）
- 中英混排，全角。英文整句中标点用半角。
## 空格
- 全角中文与半角英文间有一半角空格。
- 半角数字和百分号不留空格。
- 半角字符与全角标点不留空格。
## 第三方应用
- 标准引用格式
    > 引用文本 
- 全文引用（转载）
    > 本文转载自 [Wiki]()
- 全文+链接
    > 本文转载自 Wiki: https://en.wikipedia.org/wiki/Wiki
